### Features
1) Can Generate tables
2) Can make a Hex Table
3) Quardatic Equation Solver
4) Can Print our maths Farmulas
5) Can Convert Units
6) Can Find BMI
7) Can Evaluate Trignometric Functions
8) Can Evaluate Factorial
9) Can Evaluate basic Physics
10) Math Game
